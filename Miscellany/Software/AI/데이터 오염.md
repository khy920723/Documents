# 데이터 오염(Data Contamination)
- AI 모델을 훈련하거나 테스트하는 과정에서 의도치 않게 잘못된 데이터가 유입되거나, 테스트 데이터가 훈련 데이터에 포함되는 등의 문제를 말함.
- 이로 인해 모델이 실제 성능보다 과대 평가 되거나, 왜곡된 결과를 낼 수 있음.

# 데이터 오염의 주요 유형
## 1. 훈련-테스트 데이터 누수 (Data Leakage)
