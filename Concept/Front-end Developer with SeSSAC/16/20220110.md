> 특정 CSS속성 값의 변화를 animation 하는 메서드 (트랜지션까지 적용, CSS는 트랜지션을 직접 작성해야 됨)

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
    <style>
        div{width: 500px;height: 100px;background: red;margin-bottom: 50px;}
        #box1{transition: all 0.4s;}
    </style>
    <script src="libs/jquery-1.12.4.min.js"></script>
    <script>
        $(function(){
            $('#box1').on('click',function(){
                $(this).css({width:'1000px'}) // css에서 트랜지션 직접 작성 필요
            })
            $('#box2').on('click',function(){
                $(this).animate({width:'1000px'}) // css에서 트랜지션 직접 작성 불필요
            })
        })

## 모르는 것

## toggleClass

> toggleClass()로 선택한 요소에 클래스 값을 넣었다 뺏다 할 수 있음
>
> $('p').toggleClass('className')
>
> > p요소에 xyz 클래스가 없으면 추가하고 있으면 제거



## ::selection

> 홈페이지에서 드래그를 하는 부분의 색상을 바꿀 수 있음
>
> ```css
> <style>
> ::selection{background:#red;color:#blue}
> ::-moz-selection{background:#red;color:#blue}
> ::-webkit-selection{background:#red;color:#blue}
> </style>
> ```
>
> > -moz-, -webkit-은 브러우저들간에 같은 css를 보여주기 위해 추가되는 부분



## Hoisting (호이스팅)

> 코드가 실행하기 전 변수선언/함수선언이 해당 scope의 최상단으로 끌어 올려진 것 같은 현상

- 변수의 생성과 호이스팅 과정

1. 선언 단계(Declaration phase)

> 변수를 실행 컨텍스트의 변수 객체에 등록
>
> 이 변수 객체는 scope가 참조하는 대상이 됨

2. 초기화 단계(Initialization phase)

> 변수 객체에 등록 된 변수를 위한 공간을 메모리에 확보
>
> 이 단계에서 변수는 undefined로 초기화 됨

3. 할당 단계(Assignment phase)

> undefined로 초기화된 변수에 실제 값 할당

var 키워드로 선언한 변수는 선언 단계와 초기화 단계가 한 번에 이뤄짐
