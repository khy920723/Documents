```html
<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
  <script src="libs/jquery-1.12.4.min.js"></script>
  <script>
    $(function(){
      $(document).on('mousemove',function(e){
        // 객체.속성 -> 이벤트객체.마우스X좌표속성
        var mouseX=e.pageX;
        var mouseY=e.pageY;
        $('#box').text('마우스 X좌표: '+mouseX+'마우스 Y좌표: '+mouseY)
        var winWidth=$(window).width()/2;
        
        if(mouseX<winWidth){
          $('body').css({backgroundColor:'red'})
        }else{
          $('body').css({backgroundColor:'blue'})
        }
      }) // 문서객체에서 마우스가 움직이면(e는 mousemove로 생성 된 이벤트 객체)
    })
  </script>
</head>
<body>
  <div id="box"></div>
</body>
</html>
```

## target

- event.target

  > 이벤트가 발생한 요소를 반환
  >
  > ```javascript
  > $("a").click(function(event){
  > console.log(event.target);
  > });
  > ```
  >
  > > a요소를 클릭했을 때 이벤트가 발생하고 이벤트가 발생할 때 마다 이벤트의 대상인 a요소를 콘솔창에 출력하는 코드
  >
  > target을 이용하면 현재 이벤트가 발생한 요소의 속성들을 얻을 수 있음 (이벤트가 발생한 대상의 id등과 같은 것)
